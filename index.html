Accessibility

To make my website accessible to all users I made sure to include the language that my website is in when coding so it can be translated by screen readers.

Some things that other people used on their websites that looked cool to me were boxes around headings and columns to create unique layouts.

I am most proud of learning how to change the color and font of text and being able to put boxes around text because by doing all these things I can further customize my website.

Variables A variable is something that can be defined as a string and can be used to help make code look a bit more organized.
Loops Loops are used to repeat a section of code a certain number of times or under certain conditions.
Conditionals Conditionals are used to run a section of code only if a certain thing is true or run a different section of code if that thing is false.
